# Proyecto Flask Peso

[![Coverage Status](https://coveralls.io/repos/github/Revalyx/Flask_Peso_PPS/badge.svg?branch=main&service=github&kill_cache=1)](https://coveralls.io/github/Revalyx/Flask_Peso_PPS?branch=main)

Aplicación sencilla en **Flask** para registrar peso, visualizar historial y calcular IMC.  
Incluye autenticación básica y persistencia en SQLite.

---

## Funcionalidades
- Registro y login de usuarios  
- Registro diario de peso  
- Historial y gráfico de evolución  
- Cálculo automático de IMC  
- Actualización de altura  
- Rutas protegidas por sesión  
- Tests automatizados
